From [swyix.io](https://www.swyx.io/css-100-bytes/) who in turn cites HN's @JoshuaRLi as a source, a small snippet of CSS tobegin any style with:

```
html {
  max-width: 70ch;
  padding: 3em 1em;
  margin: auto;
  line-height: 1.75;
  font-size: 1.25em;
}
```
